# 栈

```c
int stack[maxsize];
int top = -1; //下标

stack[++top] = 1; //前++和后++ 一句话总结一下就是出栈后--，入栈先++；
temp = stack[top--];
//只是单纯的出栈
top--;

//栈判空
top == -1;
//如何判满
top == maxsize - 1;
```



# 队列

```c
int queue[maxsize];
int rear = 0; int front = 0;

//入队，出队
rear ++;
queue[rear] = temp;
queue[++rear] = temp;

//出队
font++;
temp = queue[front];
temp = queue[++front];

//队列判空,front所指向的不属于队列元素。 
front == rear;


//循环队列
//在每次出队入队中，实现了循环队列
rear = (rear+1)%maxsize;
front = (front+1)%maxsize;

//判空，判满
front == rear;
(rear+1)%maxsize == front;
```
